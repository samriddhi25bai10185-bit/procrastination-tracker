Procrastination Tracker

📌 Description
This is a command-line Python project that helps users track their daily study habits and analyze productivity. The user enters planned study hours and actual study hours, and the system calculates a productivity score. It provides daily feedback, highlights the best and worst day, and helps users reduce procrastination over time.

🚀 Features
Add daily study data
View saved records
Calculate productivity score
Show analysis (average score, best day, worst day)
Time wasted warning for low productivity
Clear all stored data
Simple and interactive CLI menu

🎯 Target Audience
This project is designed for:
Students who want to track their study habits
Beginners learning Python and basic project development
Individuals trying to reduce procrastination
Anyone interested in improving productivity and time management

🛠️ Technologies Used
Python
JSON (for data storage)

▶️ How to Run
Open a terminal in the project folder.
Run the program:
python main.py
Follow the menu options:
--- Procrastination Tracker ---
1. Add today's data
2. View records
3. Show analysis
4. Clear all data
5. Exit

📊 Example
Enter planned study hours: 5
Enter actual study hours: 3
📉 Goal not achieved today
⚠️ You wasted a lot of time today. Try to focus more!
Score: 60%
🙂 Not bad, try to reduce distractions.
